<h2 align="center">My first Laravel CRUD</2>
Learning Laravel and Bootstrap

This project is a very simple CRUD that allows you to register products.

To create the database and the products table it is only necessary to correctly configure the MySQL server authentication data and click on the options for these functions found in the navigation bar, and then you can register products.

It also has two simple applications developed with JavaScript. A sales calculator and a computer unit converter.
